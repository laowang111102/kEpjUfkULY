# 旅游景点推荐系统 python815

## 项目概述

旅游景点推荐系统是基于 Python 的 Django 框架开发的，旨在提供景点信息管理、旅游咨询发布、行程分享及个性化推荐等功能。

## 技术栈

- **后端技术**：Python、Django 框架、MySQL 数据库
- **前端技术**：Vue
- **开发工具**：PyCharm

## 功能模块

### 前台管理

- 景点信息
- 旅游咨询
- 行程分享
- 个人中心
- 我的收藏

### 后台管理

- 景点信息管理
- 景点分类管理
- 酒店信息管理
- 行程分享管理
- 系统管理

## 系统角色

系统包含前台用户和后台管理员两种角色。

- **前台用户**：浏览景点信息，发布行程分享，收藏景点等。
- **后台管理员**：管理景点信息，分类，酒店信息，以及系统设置。

## 运行环境

- Python 3.8+
- Django 3.2+
- MySQL 5.7+
- Vue.js 2.x+

## 部署步骤

1. 环境准备：安装 Python、MySQL、Node.js 等。
2. 克隆项目代码到本地。
3. 根据配置文件配置数据库。
4. 安装项目依赖。
5. 运行项目。
